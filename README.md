learn-automata
==============

Collection of automata-related algorithms for student, including:

1. Recognizing words
    - DFA
    - NFA
    - NFA-e
    
2. Converting
    - NFA-e to NFA
    - NFA to DFA
    
3. Checking properties
    - Universality
    - Emptiness
    
4. Operating 2 automata
    - Complement
    - Union
    - Intersection
    - Symmetric Difference

5. Minimizing
    - Hopcroft algorithm
    - Moore algorithm
    - Brzozowski algorithm
